## The Three Pillars

The three pillars of writing good HTML and CSS

- Responsive design

  - Fluid layouts
  - Media queries
  - Responsive images
  - Correct units
  - Desktop-first vs mobile-first

- Maintainable and scalable code

  - Clean
  - Easy-to-understand
  - Growth
  - Reusable
  - How to organize files
  - How to name classes
  - How to structure HTML

- Web performance

  - Less HTTP requests
  - Less code
  - Compress code
  - Use a CSS preprocessor
  - Less images
  - Compress images
